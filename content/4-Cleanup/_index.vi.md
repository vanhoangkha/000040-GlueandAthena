#### Dọn dẹp tài nguyên

#### Xóa Crawler của AWS Glue

1. Truy cập giao diện **AWS Glue**
2. Chọn **Crawlers**
3. Chọn crawler liên quan bài lab
4. Chọn **Actions** và chọn **Delete crawler**
5. Chọn **Delete**

![Clean up](/images/4-Cleanup/0001-cleanup.png?featherlight=false&width=90pc)

#### Xóa database trong AWS Glue

1. Truy cập giao diện **AWS Glue**
2. Chọn **Databases**
3. Chọn database liên quan bài lab
4. Chọn **Delete**

![Clean up](/images/4-Cleanup/0002-cleanup.png?featherlight=false&width=90pc)

#### Xóa S3 bucket

1. Truy cập giao diện **S3**
2. Chọn **Bucket**
3. Chọn bucket liên quan bài lab
4. Chọn **Empty**

![Clean up](/images/4-Cleanup/0003-cleanup.png?featherlight=false&width=90pc)

5. Nhập **permanently delete** và chọn **Empty**

![Clean up](/images/4-Cleanup/0004-cleanup.png?featherlight=false&width=90pc)

6. Chọn bucket liên quan bài lab và chọn **Delete**

![Clean up](/images/4-Cleanup/0005-cleanup.png?featherlight=false&width=90pc)

7. Xác thực tên bucket và chọn **Delete bucket**

![Clean up](/images/4-Cleanup/0006-cleanup.png?featherlight=false&width=90pc)
